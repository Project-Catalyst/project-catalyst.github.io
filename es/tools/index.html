# Tools

Cardano Updates
cardanoupdates.com
This is a technical update tracker for the Cardano (ADA) project. It aggregates commits within the last 7 days from all branches of Cardano development-related repos using local git mirrors. The same content can also be acquired from GitHub's web interface.
